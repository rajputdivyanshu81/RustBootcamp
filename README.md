Installation

To get started with the projects and exercises in this repository, clone it to your local machine using the following command:


git clone https://github.com/yourusername/rust-bootcamp.git


Navigate to the directory:


cd rust-bootcamp

Usage

You can explore the different projects and exercises by navigating to the respective directories. Each directory contains the source code and instructions on how to run the programs.

For example, to run the Guessing Game project:


cd projects/guessing_game

cargo run
